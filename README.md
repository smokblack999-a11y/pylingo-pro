# PyLingo Pro

Python IDE + Remote Control - образовательная платформа с удалённым доступом

## Возможности

- 🐍 Python в браузере (выполнение кода)
- 📚 Интерактивный курс Python с XP
- 🌐 Удалённый доступ через WebRTC
- 🔒 Безопасная песочница
- ⚡ Работает в Termux

## Установка

```bash
# Установка зависимостей
pkg install python
pip install flask

# Запуск
cd app
python main.py
```

Открой http://localhost:8000
